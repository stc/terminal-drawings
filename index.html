Terminal Drawings

Algorithmic plotter artworks, created with a command-line interface

How would one represent addressable space? Given that digital information doesn't inherently exist in physical space, we can use any structure to illustrate relationships between files, folders, thoughts and other conceptual entities. The artwork involves text based simulations to represent such spaces. As the computer terminal is a low-level ontological tool for interfacing the real world with computational landscapes, the series resembles the performative mechanics of such gateways.
